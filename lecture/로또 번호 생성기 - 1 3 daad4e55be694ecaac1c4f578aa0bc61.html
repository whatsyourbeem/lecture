React Native 프로젝트 생성

React Native(이하 RN) 프로젝트를 생성 해 보겠습니다.

저는 주로 C드라이브에 workspace 폴더에서 프로젝트를 생성하고 있습니다.

윈도우 시작버튼 누른 후 PowerShell을 입력해서 PowerShell을 띄운 후에 아래 명령으로 프로젝트를 생성 합니다.

C:\workspace> npx react-native init RnLottoNumGen

별 에러 없이 생성이 완료 되면 VS Code를 이용해서 C:\workspace\RnLottoNumGen폴더를 엽니다.

새로 생성한 프로젝트 테스트

새로 생성한 프로젝트가 스마트폰에서 잘 실행되는지 한번 테스트 해보겠습니다.

RN 프로젝트를 생성하면 실행 가능한 기본 RN 앱이 생성 됩니다.

RN 기본 앱을 폰에서 실행하려면 두가지 필요한 작업이 있습니다.

  1. 폰과 컴퓨터 연결
  1. RN 프로젝트에서 안드로이드 Run 명령 실행

폰과 컴퓨터 연결

안드로이드 폰에서 설정>개발자 옵션>USB 디버깅 설정 해주시고

USB로 컴퓨터와 스마트폰을 연결 한 후

VS Code에서 터미널을 열고(단축키: Ctrl+`) 아래의 명령을 입력합니다.

C:\workspace\RnLottoNumGen> adb reverse tcp:8081 tcp:8081

그러면 스마트폰 화면에 아래와 같이 뜰꺼에요. 허용 해 줍니다.

자~! 이제 RN 프로젝트에서 안드로이드 Run 명령을 실행해 보시죠~!

RN 프로젝트에서 안드로이드 Run 명령 실행

VS Code 터미널에서 아래와 같이 명령을 실행 합니다.

C:\workspace\RnLottoNumGen> npx react-native run-android
