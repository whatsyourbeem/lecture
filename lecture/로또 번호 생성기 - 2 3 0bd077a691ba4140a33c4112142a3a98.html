요소(Component) 추가하기

“틀 작성” 부분에 계산기의 요소들(텍스트, 버튼)들을 코드로 정의 할 꺼에요.

텍스트와 버튼은 아래와 같은 형태로 “틀 작성” 부분에 넣으시면 됩니다.

<Text> 표현할 값 </Text> // 텍스트 요소 추가 방법
<Button title='버튼 위 글자'></Button> // 버튼 추가 방법

요소(Component) 정렬하기

화면에 11, 22, 33, 44, 55, 66가 세로로 나왔는데, 가로로 나오면 좋을 것 같아요. 가로로 나오도록 수정해 보겠습니다. Text들을 View로 감싼 다음에 정렬 관련 스타일을 추가해 줍니다.

View의 스타일 내용을 요약하면,

  • View가 화면의 가로 길이(width)를 100% 다 쓰고, (width: '100%',)
  • 마진은 20정도 두고, (margin: 20,)
  • View 내부에 있는 Text들을 가로 형태로 정렬 하면서 (flexDirection: 'row',)
  • View 내부 요소들이 서로 간격을 같게 유지하도록 하는 것 (justifyContent: 'space-around')

입니다.

import React from 'react';
import { View, StyleSheet } from 'react-native';

const App = () => {

  return (
    <View>

			<View style={{
        width: '100%',
        margin: 20,
        flexDirection: 'row', // 가로로 정렬
        justifyContent: 'space-around' // 맞춤 방식 설정
      }}>
        <Text>11</Text>
        <Text>22</Text>
        <Text>33</Text>
        <Text>44</Text>
        <Text>55</Text>
        <Text>66</Text>
      </View>
			<Button title='Generate'></Button>

    </View>
  )
};

const styles = StyleSheet.create({

});
export default App;

6개의 숫자와 버튼이 너무 상단에 있으니, 중앙으로 내려 보겠습니다.

가장 바깥에서 감싸고 있는 View에 스타일을 적용합니다.

View의 스타일 내용을 요약하면,

  • View가 화면의 세로 길이(height)를 100% 다 쓰고, (height: '100%',)
  • View 내부의 각 아이템들을 가로방향 가운데 정렬 (alignItems: 'center',)
  • 세로방향 가운데 정렬 (justifyContent: 'center')
import React from 'react';
import { View, Text, Button, StyleSheet } from 'react-native';

const App = () => {

  return (
    <View style={{
      height: '100%',
			alignItems: 'center',
      justifyContent: 'center'
    }}>

      <View style={{
        width: '100%',
        margin: 20,
        flexDirection: 'row', // 가로로 정렬
        justifyContent: 'space-around' // 맞춤 방식 설정
      }}>
        <Text>11</Text>
        <Text>22</Text>
        <Text>33</Text>
        <Text>44</Text>
        <Text>55</Text>
        <Text>66</Text>
      </View>

      <Button title='Generate'></Button>
    </View>
  )
};


const styles = StyleSheet.create({

});
export default App;

스타일 수정하기

요소들을 추가, 정렬 했으니, 이젠 요소들의 스타일(동그라미 테두리)를 적용 해 보겠습니다.

위에서 요소들을 정렬 할때, 해당 요소 상위에 View를 두고 View의 style을 수정하는 것을 보셨을 겁니다.

지금은 각 Text의 동그라미 테두리를 만들어 줄 것이기 때문에, 각 Text를 View로 감싸고, 테두리를 적용 하겠습니다.

그 전에, 각 테두리에 적용할 스타일을 먼저 정의 하겠습니다. 코드에서 하단에 있는 “스타일 작성” 부분에 아래와 같이 numBorder추가합니다.

const styles = StyleSheet.create({
	numBorder: {
    width: 40,
    height: 40,
    borderRadius: 40,
    borderWidth: 1,
    justifyContent: 'center',
    alignItems: 'center'
  },
});
export default App;

그리고 각 Text를 View로 감싸고, View의 style에 numBorder를 추가해 줍니다.

import React from 'react';
import { View, Text, Button, StyleSheet } from 'react-native';

const App = () => {

  return (
    <View style={{
      height: '100%',
      alignItems: 'center',
      justifyContent: 'center'
    }}>

      <View style={{
        width: '100%',
        margin: 20,
        flexDirection: 'row', // 가로로 정렬
        justifyContent: 'space-around' // 맞춤 방식 설정
      }}>
        <View style={styles.numBorder}>
          <Text>11</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>22</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>33</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>44</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>55</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>66</Text>
        </View>
      </View>

      <Button title='Generate'></Button>
    </View>
  )
};


const styles = StyleSheet.create({
  numBorder: {
    width: 40,
    height: 40,
    borderRadius: 40,
    borderWidth: 1,

  },
});
