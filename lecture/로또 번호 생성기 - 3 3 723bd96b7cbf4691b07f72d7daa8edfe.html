이번에는 화면에 있는 요소들이, 버튼을 눌렀을 때 새로운 로또 번호를 표시하도록 로직 달아 보겠습니다.

값을 기록할 공간 만들기

숫자 6개를 랜덤으로 뽑은 후 화면에 보여주기 위해선, 뽑은 6개의 숫자를 “잠시 어딘가”에 기록해 두고, 그 숫자들을 화면에 표시하도록 해야 합니다. 이때 사용할 “잠시 어딘가”를 코드상에 만들어 보겠습니다.

지금은 앱을 키면 첫 화면으로 11, 22, 33, 44, 55, 66가 표시되고 있는데, 첫 화면에선 “L O T T O !”가 표시되도록 “잠시 어딘가”의 초기값을 설정하겠습니다.

<Text>{numArray[0]}</Text>의 내용은

  • Text안에 값(11, 22 같은) 대신 중괄호로 시작해서, 중괄호 안에 있는 글자는 그대로 표현하는게 아니라 코드로 생각해 줘! 라는 뜻이고,
  • numArray라는 공간이 로직부분에 만들어져 있어서 여기에서 쓸꺼고,
  • 그 numArray라는 공간의 제일 첫번째인 [0]자리의 값을 여기 Text에 표시해줘! 라는 뜻 입니다.(코딩에서는 1이 아니라 0을 순서의 첫번째로 인식하는 경우가 많습니다)

버튼을 눌렀을 때 numArray의 값들을 랜덤으로 수정하기

“로직 작성” 부분에 gen을 만듭니다.

gen에는 GENERATE 버튼을 눌렀을 때 실행할 작업을 정의 합니다. (로직 작성 부분에 아래 코드를 추가 해주세요)

const gen = () => {
    const tempNumArray= []; // 임시로 쓸 공간을 하나 만들고(tempNumArray)
    while (tempNumArray.length < 6) { // 공간의 값이 6개보다 적게 모였을 때
        let num = Math.floor(Math.random() * 45) + 1; // 1~46 중에 하나를 랜덤으로 뽑아서
        if (tempNumArray.indexOf(num) === -1) { // tempNumArray에 있는 값과 중복되는 숫자가 없을시에
            tempNumArray.push(num)// 랜덤으로 뽑은 값을 lottoNumbers에 넣어 줍니다.
        }
    }
    setNumArray(tempNumArray); // tempNumArray에 값이 6개 모였으니 화면에 표시할 공간인 numArray에 넣어줍니다.
}

그리고 Button에 onPress로 방금 만든 gen 작업을 할수 있도록 연결해 줍니다.

전체 완성된 코드는 아래와 같습니다.

import React, { useState } from 'react';
import { View, Text, Button, StyleSheet } from 'react-native';

const App = () => {
  const [numArray, setNumArray] = useState(['L', 'O', 'T', 'T', 'O', '!']);

  const gen = () => {
    const tempNumArray= []; // 임시로 쓸 공간을 하나 만들고(tempNumArray)
    while (tempNumArray.length < 6) { // 공간의 값이 6개보다 적게 모였을 때
        let num = Math.floor(Math.random() * 45) + 1; // 1~46 중에 하나를 랜덤으로 뽑아서
        if (tempNumArray.indexOf(num) === -1) { // tempNumArray에 있는 값과 중복되는 숫자가 없을시에
            tempNumArray.push(num)// 랜덤으로 뽑은 값을 lottoNumbers에 넣어 줍니다.
        }
    }
    setNumArray(tempNumArray); // tempNumArray에 값이 6개 모였으니 화면에 표시할 공간인 numArray에 넣어줍니다.
}

  return (
    <View style={{
      height: '100%',
      alignItems: 'center',
      justifyContent: 'center'
    }}>

      <View style={{
        width: '100%',
        margin: 20,
        flexDirection: 'row', // 가로로 정렬
        justifyContent: 'space-around' // 맞춤 방식 설정
      }}>
        <View style={styles.numBorder}>
          <Text>{numArray[0]}</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>{numArray[1]}</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>{numArray[2]}</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>{numArray[3]}</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>{numArray[4]}</Text>
        </View>
        <View style={styles.numBorder}>
          <Text>{numArray[5]}</Text>
        </View>   
      </View>

      <Button title='Generate' onPress={gen} />
    </View>
  )
};


const styles = StyleSheet.create({
  numBorder: {
    width: 40,
    height: 40,
    borderRadius: 40,
    borderWidth: 1,
    justifyContent: 'center',
    alignItems: 'center'
  },
});
export default App;
